# 介绍

MySQL索引可以大大提高MySQL的检索速度。

分单列索引和组合索引。

  • 单列索引，一个索引只包含单个列，一个表可以有多个单列索引，但这不是组合索引；
  • 组合索引，即一个索引包含多个列。

索引也会有它的缺点：

  • 降低更新表的速度，因为MySQL不仅要保存数据，还要保存一下索引文件。
  • 占用磁盘空间

创建索引时，你需要确保该索引是应用在 SQL 查询语句的条件(一般作为 WHERE 子句的条件)。
