## 常用命令
- git init : 本地项目初始化
- git add : 编辑区添加到暂存区
- git commit : 暂存区提交到分支
- git push : 同步到远程仓库
- git remote : 创建远程主机名
- git clone : 克隆项目到本地
- git pull : 拉取项目代码
- git branch : 查看分支
- git branch name : 创建分支
- git checkout : 切换分支
- git merge : 合并分支
- git log : 查看历史记录
- git reset : 恢复版本
- git reflog : 查看命令记录
