# 费曼学习法

  1. 选择一个概念，进行系统学习。
  2. 讲授这个概念。给其它人讲解这个概念，并让对方完全听懂，这可以加深你的理解。
  3. 查漏补缺。
  4. 简化语言和尝试类比。
